# Custom weapons showcase
![visitors](https://visitor-badge.glitch.me/badge?page_id=talesrune.customweapons)\
To obtain one of these custom weapons, simply type **!c** or **/c**.
 
# Commands
- **sm_c** - Give the user a menu to select his/her custom weapons.

# Installation
**Your server needs both [TF2Items](https://builds.limetech.org/?p=tf2items), [TF2Attributes](https://forums.alliedmods.net/showthread.php?t=210221), and [TF2 Custom Weapons 2](https://github.com/404UNFca/CustomWeapons2) loaded!**
* Unzip customweapons zip file.
* Drag and drop customweapons folder into your sourcemod/configs/ directory.
* Done!

# Directory
* configs folder - 1. customweapons folder - 1. All txt files that contains custom weapons
